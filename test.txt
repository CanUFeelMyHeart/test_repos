# Работа с репозиториями GitHub

1. Чтобы начать работу с каким-либо удаленным репозиторием другого автора, необходимо:
* совершить операцию Fork на Github - будет создан удаленный репозиторий на своём аккаунте со ссылкой на оригинал
* выполнить команду git clone <url ссылка в формате https://github.com...> - удаленный репозиторий будет скачан в локальную папку
* после редактирования выполнить команду git push - загрузка изменений в удаленный репозиторий

2. Чтобы предложить изменения автору проекта, необходимо на платформе Github отправить запрос кнопкой "New pull request":
* указать тему обращения
* добавить описание изменений
