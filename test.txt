# Инструкция по работе с Git
В данной инструкции мы разберем самые базовые команды **Git**, которые необходимо знать **каждому**.

![Top_Git_Commands](https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcSJCnqhrR-JOQD4_XcH-UTnYLiofjv9CtYN9Q&usqp=CAU.jpg)

1. Для того, чтобы *создать пустой репозиторий*, нужно:

    >git init

2. Чтобы начать *сохранение файла* с именем file_name, нужно:  
    > git add file_name или git add .

3. *Создание коммита*: (комментарий для программиста)  
    >git commit -m message "your_text"

4. Чтобы показать *журнал всех коммитов*, нужно:  
    >git log

5. Чтобы *вернуться к какому-либо коммиту*, нужно:  
    >git checkout <хэш-код>  
    >git checkout master (Вернуться к последней версии)

6. Чтобы увидеть *разницу между текущим файлом и закомиченным файлом*, нужно:  
    >git diff
7. Чтобы вывести все ветки на экран, мы используем команду:
    > git branch
8. Чтобы создать новую ветку, мы используем команду:
    + git branch <branch_name>
    
## Новые команды:
* git commit -a -m 'message' - сохарнение текущего файлы и коммит
+ git checkout -b <bracnh_name> - создание новой ветки и автоматический переход в нее

Чтобы перейти к другой ветке, мы используем: 
1. git checkout <название ветки>

7. Чтобы удалить ветку с __замерженными изменениями__, мы используем команду:
* git branch -d <branch_name>
8. Чтобы удалить ветку с __незамерженными изменениями__, мы используем:
* git branch -D ,branch_name>
## Команды для работы с удаленным репозиторием
1. Для того, чтобы сделать клон (скопировать) внешний репозиторий с с GitHub на локальный ПК, нужно ввести команду:
> git clone <url-адрес репозитория> 

2. Чтобы выкачать изменения с удаленного репозитория и слить его с локальной версией, нужно:
> git pull

3. Чтобы отправить локальную версию репозитория на внешний репозиторий, нужно:
> git push

# Синтаксис языка Markdown

> Чтобы создать таблицу с помощью языка markdown, можно использовать вертикальные и горизонтальные линии следующим образом: 

|Страна      |Столица     |Местное время|
|------------|------------|-------------|
|Россия      |Москва      |16:13        |
|Япония      |Токио       |22:14        |

>Чтобы сделать текст зачеркнутым, нужно обрамить его двойным знаком тильда (~~).
Например: ~~зачеркнутый текст~~
Чтобы отедилть один текст от другого чертой, нужно на следующей строке после текста поставить знак "---". 
 
Например: текст 1

---

текст 2
