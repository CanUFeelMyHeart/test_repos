# Команды для GitHub

1. git clone - эта команда позволяет скачать репозиторий
на свой ПК
2. git push - эта команда позволяет отправить свой репозитории на github
3. git pull - Эта команда позволяет скачать все из текущего репозитория и автоматически
сделать merge с нашей версией
